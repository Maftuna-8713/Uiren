HTML

Web парақша құру

HTML("HyperText Markup Language")-гипермәтінді белгілеу тілі.HTML тілін 1991 жылы Женевадағы физикалық зерттеулер институты қызметкері Тим Бернерс-Ли таралған гипермәтіндік жүйе жасау жобасын жүзеге асыру кезінде жасап, оны World Wide Web немесе Жаһандық өрмек деп аталады.

HTML файдарын жасауға арналған көптеген редакторлар бар, соның біреуі Sublime Text редакторы.

Бірінші Web парақшаны құруды бастайық:

1.html папкасын құрамыз

2.Папканың ішіне мышканың оң жақ батырмасын басып мәтіндік құжат құрамыз

3.Мәтіндік құжатты index.html деп өзгертеміз

Төменде көрсетілген видеоны қараңыз



Тәгтермен танысу

HTML құжаты құрамында арнайы белгілеу тәгтері (немеме басқаратын элементтері) бар кәдімгі мәтіндік файл болып табылады.Тәгтер Web браузерге (web-парақтарды көрсетуге арналған программа, мысалы, Internet Explorer, Mozilla,Netscape Opera)парақтағы мәтінді, суретті экранға қалай шығару керектігін көрсетеді. Тәг символлар тізбегінен тұрады.Кез келген тәг "кіші" (<) символынан басталады да, "үлкен" (>) символымен аяқталады.Осындай символ бұрыштық жақшалар деп те аталады.Ашылатын бұрыштық жақшадан соң команда аты болып табылатын ағылшын тіліндегі түйінді сөз-тәг орналасады. Тәгтердің ашатын тәг және жабатын тәг түрлері болады.Жабу тәгтері қиғаш сызық символымен(/)басталуы тиіс.


Негізгі тәгтер

HTML құжатының кез келгені <HTML> тәгінен басталып, соған сәйкес </HTML>түрінде жабылу тәгімен аяқталады.Осы екеуәнің ортасында құжаттың тақырыптық бөлігі мен тұлғасы болып келетін негізгі бөлігі орналасады.

Құжаттың тақырыптық бөлігі <HEAD> және </HEAD> тәгтерінің ортасында тұрады да,жалпы құжат туралы мәлімет береді.Әдетте, бұл бөлікте <TITLE>....</TITLE> тәгтерімен шектелетін құжаттың терезе маңдайшасында тұратын ресми атауы орналасады.Көптеген браузерлер оны терезе тақырыбында тұратын файл фты есебінде пайдаланылады.

Жазылатын мәтін құжаттың негізгі тұлғасы деп аталатын <BODY>....</BODY>тәгтерінің ортасында жазылады.Бұл қос белгі HTML-құжаттың негізгі мазмұндық бөлігінің басын және соңын білдіреді.

HTML тілінің комментарийлерін тәг жазбай-ақ, арнайы символдардан кейін жалғастырып жазуға болады.Түсініктеме мәтін соңына символдары жазылуы тиіс.Төменде көрсетілген видеоны қараңыз.


Кесте құру

Кесте тұрғызу <Table> және </Table> тәгтері көмегмен орындалады, оның әрбір жолын анықтау -<TR> және </TR>тәгтері арқылы, ал сол жолдағы бағаналар - <TD> және </TD> немесе <TH> және </TH> тәгтері арқылы анықталады.<TD> және <TH> тәгтерінің жұмысы ұқсас, бірақ <TH> тәгтермен қоршалған мәтін қарайтылған бағана тақырыптары болып табылады да, <TD> тәгтерімен одан кейінгі қарапайым бағаналар жазылады. Кесте тақырыбы <Caption> және </Caption> тәгтерімен қоршалып жазылады.Төменде көрсетілген видеоны қараңыз.


CSS құжатын құру

CSS - Cascading Style Sheets(Стильдердің сатылы кестелері)-гипермәтіндік мәліметтер мазмұнын олардың экрандағы бейнелену формасынан бөліп орындауға мүмкіндік беретін құрал болып табылады.CSS стильді қосудың 2 тәсілі бар.Бірінші тәсіл:<BODY>....</BODY>тәгтерінің ортасында <STYLE> және </STYLE> тәгінің ішіне жазылады.Төмендегі видеода толық ақпарат берілген.Екінші тәсіл: жаңа css құжатын құру және оны сілтеме арқылы HTML құжатқа қосу.Қасиетін өзгертетін тәгтер css құжатының ішіне жазылады.Төмендегі видеода толық ақпарат берілген.Төменде көрсетілген видеоны қараңыз.


CSS қасиеттеріне шолу

Стильдегі оның қасиеттерін мынадай топтарға бөліп беру қалыптасқан:қаріп(шрифт),түстер, мәтін, өрістер мен жақтаулар (поля и рамки), сыртқы түрлері.Шрифтер үшін: "font-family" - қаріп түрі.Бұл стильдік қасиет шрифтің гарнитурасы атын немесе оның топ атауын көрсетеді. "font-size"-қасиеті шрифтің абсолюттік немесе салыстырмалы мөлшерін береді.Салыстырмалы түрде бергенде, пайыздық өлшем қолданылады(ағымдық шрифт негізгі) немесе мынадай түйінді сөздер қою керек:larger-ірілеу,smaller-шағындау.Абсолюттік мөлшерде келесі өлшем бірліктерінің бірін көрсету керек:in, cm, mm, px, pt, pc.Түс. Стильдік қасиеттер түстерді тағайындаудың үш түрлі тәсілін береді: түйінді сөз(white) , он алтылық RGB-код(#eee5d8) , ондық RGB-код(rgb(255,64,0)).Color-элемент түсін анықтайды.background-color-элемент шығарылатын маңайдағы фон түсін анықтайды.Мәтін үшін. text-align-мәтінді көлденең туралау тәсілін көрсетеді.Келесі мәндер қолданылады:left-сол жақ шетке туралау, right-оң жақ шетке туралау, center-ортаға туралау, justify-екі шетінде туралау.Төменде көрсетілген видеоны қараңыз.


CSS Сурет қою

CSS құжатында сурет қою үшін background-image қасиетін қолданамыз.Суреттің енін өзгерту үшін width қасиетін, ұзындығын өзгерту үшін height қасиеттерін қолданамыз.Суреттін орналасуын беру үшін align қасиетін қолданамыз.Төмендегі видеода толық ақпарат берілген.Төменде көрсетілген видеоны қараңыз.


JavaScript құжатын құру

JavaScript программалау тілін Netscape пен Sun Microsystems фирмасы бірлесіп ұсынған, ол интерактивті HTML-құжаттарды құруға арналған.JavaScriptтілінде жазылған программалар операторлар тізбегінен тұрады.JavaScript тілінде бас әріптер мен кіші әріптер әр түрлі символ болып саналады.JavaScriptпен жұмыс істеудің екі түрлі тәсілі бар.Біріншісі-<SCRIPT> және </SCRIPT> тәгтері арқылы HTML құжатына тікелей жазылады.Екінші тәсіл-js құжатын құрып, соның ішіне жазамыз.Ол үшін HTML құжатына сілтеме арқылы байланыстырамыз.Төменде көрсетілген видеоны қараңыз.


Айнымалылар

Айнымалылар ақпараттарды сақтау үшін қолданылады. Айнымалылар сценарийлерде иденфикаторлардың көмегі арқылы көрсетіледі. Иденфикатор латын әріптерінен, цифрлардан, астын сызу белгісінен басталуы керек, мысалы, test_1.Айнымалының типі сценарийде сақталған ақпараттың типіне байланысты болып, сол мәліметтің типі өзгерсе, айнымалының типі де өзгереді.Айнымалы var операторы арқылы анықталуы тиіс, мысалы: var test1.Мұнда test1 айнымалысының типі әлі анықталмаған, ол айнымалыға белгілі бір мән меншіктеген кезде ғана айқындалады.var операторын айнымалыны инициализациялағанда да қолдануға болады, мысалы:var test2=276.Мұнда test2 айнымалысының типі анықталды және ол 276 мәніне тең болды.Төменде көрсетілген видеоны қараңыз.


Өрнектер

Өрнектер операциялар (амалдар) таңбаларымен біріктірілген литералдан, айнымалылардан және жақшалардан тұрады.Өрнектерді есептеу нәтіжесінде сан(бүтін не нақты), сөз тіркесі немесе логикалық типте болатын бір ғана мән шығады.Өрнектерде қолданылатын айнымалылар оған дейін инициалданған болу керек.Өрнектерді есептеу кезінде анықталмаған немесе инициалданбаған айнымалылар кездессе қате шығады.

Өрнек операндтардан(мәндер мен айнымалылардан) және операциялар таңбаларынан(+, -, *,/)тұрады.Мысалы, a*b формуласында a және b операндтар, * таңбасы-көбейту операциясы.Төменде көрсетілген видеоны қараңыз.
